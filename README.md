#  AI-Based Offline Exam Monitoring System

##  Description

A Python-based system that uses computer vision to monitor students during offline exams. It detects faces in real-time and triggers an alarm when suspicious activity (no face, multiple faces, etc.) is detected.

---

##  Features

* Real-time face detection
* Detects absence or multiple faces
* Alarm for suspicious behavior
* Works offline

---

##  Tech Stack

* Python
* OpenCV
* NumPy

---

##  Usage

```bash
python main.py
```

---

##  How It Works

* Captures video via webcam
* Detects faces using OpenCV
* Triggers alert on rule violations

---

##  Future Scope

* Face recognition
